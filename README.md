## Design patters

Would like to see a implementation of an design patterns ? This repository has a purpose to store design patterns, with diagrams to explain and code examples.

The repository is organized as follow:

```
|-- name of desing pattern
    |-- programing language
        |-- diagram
        |-- example of implementation 1
        |-- example of implementation 2
```

Let's use PHP as example, so we are going to implemente the Factory Pattern

```
|-- Factory Pattern
    |-- PHP
        |-- diagram.md
        |-- SimpleFactory.php
        |-- RealWOrldFactoryImplementation.php
```
